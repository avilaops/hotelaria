# Screenshots

Adicione capturas de tela da aplicação nesta pasta para uso no README.md

## Imagens sugeridas:

1. **dashboard.png** - Tela principal com estatísticas
2. **reservas.png** - Listagem de reservas
3. **quartos.png** - Cards de quartos
4. **hospedes.png** - Gestão de hóspedes
5. **calendario.png** - Calendário de disponibilidade
6. **financeiro.png** - Relatórios financeiros

## Como adicionar ao README:

Substitua as URLs de placeholder por URLs reais das imagens:

```markdown
![Dashboard](./docs/screenshots/dashboard.png)
![Reservas](./docs/screenshots/reservas.png)
```

Ou use URLs do GitHub:

```markdown
![Dashboard](https://raw.githubusercontent.com/avilaops/hotelaria/main/docs/screenshots/dashboard.png)
```
